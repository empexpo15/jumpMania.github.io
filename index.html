Project

Jump Mania is a platform-style game focused on movement, collectibles, and player skill. The goal of the game is to navigate levels, collect coins, avoid hazards, and reach the endpoint.

Development

I started developing this game by first learning how Unreal Engine works and how to use it, and I did this by creating my own mini project that was a simple FPS.

I then started developing the game and first began by setting up the coin and collectable mechanic.

Next I worked on getting the player’s health and lives system working, which took a bit of a challenge, but thankfully I got it all working. After that, I added player hazards, the endpoint of the game, and finally completed all of the UI to reach the final version of the game.
